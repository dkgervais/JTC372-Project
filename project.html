Alexa Air Quality Application

In Spring of 2021, Danielle took part in a semester-long project to create an Alexa application utilizing a Raspberry Pi. The Raspberry Pi communicated with a connected air quality breakout board in order to gauge the indoor air quality. After gauging the air quality, the Raspberry Pi communicated with an attached Alexa device on command in order to indicate the user of the current air quality!
